Introduction

Full Product Description

Jade Palace is a business hotel resort located in Panama City Beach, Florida that is dedicated to providing an experience that customers will never forget. Jade Palace is the first resort in the world that is designed solely for party goers. Our priority is to ensure that our visitors get the ultimate party experience. To register gor this resort, users can call by phone or by applying online.

Test Objectives

Users should be able to access all hyperlinks. Users should be unable to view any breakouts in the webpage

Task performance

  • Hyperlink Structure
    • User must access to all links
  • View Format Structure
    • User must be able to find everything easily
  • View Coloring Structure
    • User must see if the color design blends together nicely
  • View Imaging Structure
    • User must see if the images work nicely with the Product

Experimental Design

The goal achievement efficiency was calculated for the two tasks:

  • The user was able to successfully open the link
  • The user was able to successfully view all pages

Usability Metrics

  • Effectiveness
    • This is measured by the maximum time a user must reroute to get from one page to the next
  • Efficiency
    • This is measured by how quickly users can view all pages
  • Satisfaction
    • This is soley measured by how much a user liked the webpage as a whole
