Conceptual Design

The conceptual design explains the hierarchy of the website and the basic flow. It lists the general features available in the website (pages to be visited).

Website Plan

The website plan shows all the media types to be used in each page in the website. Each page is assumed by default to use HTML and CSS. HTML forms, JavaScript, and other multimedia are some of the media types that are shown in the website plan.

Hyperlink Structure

The hyperlink structure is similar to the conceptual design in the sense that it shows all pages that can be accessed in the website, but it further shows the links between each one of them (i.e. which page takes you where). By providing the hyperlink structure, it also shows how many steps it will require to get to a specific part of the website.

Media Types

  • Text/Html
    • Used as an interface between the user and the project.
    • Created using text editors such as Notepad, Notepad++, Sublime Text Editor, etc.
  • Text/Css
    • Used for layout design and visualization to display the pages of the project.
    • Created using text editors such as Notepad, Notepad++, Sublime Text Editor, etc.
  • Text/Javascript
    • Used to add more advanced functionality to the html page.
    • Created using text editors such as Notepad, Notepad++, Sublime Text Editor, etc.
  • Images/PNG
    • Used to provide more visualization that represent the project.
