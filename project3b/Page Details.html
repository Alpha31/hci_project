Hotel Reservations Page Details

The hotel reservations page will allow guests of the website to search the rooms offered at the resort and select the room of their choice. There will be a multitude of rooms to select from and these rooms will vary in price and features such as the number of beds, kitchens, ocean views, giving users plenty of flexibility in regards to options to pick from. Information on the specific details of each and every room will also be offered on this page. This page will contain images (.jpeg, .gif, .png), radio buttons and checkboxes, HTML form, CSS for styling, hyperlinks to other pages, javascript, hyperlinks through images. This page will not feature video, html table, flash or shockwave, java applet, text formatting, text formatting and text breaks, or html frames.

Schedule Page Details

The schedule page will allow guests of the website to schedule a stay at the resort. At this point in the process, the guest has already selected a room and is ready to select when and for how long they will stay at the resort. The schedule page allows guests to carry out this portion of the process. This page will contain images (.jpeg, .gif, .png), radio buttons and checkboxes, HTML form, CSS for styling, hyperlinks to other pages, Html table, javascript. Album Page Details

The Album page will serve as a platform that gives a guest of the site a good idea of what the resort has to offer. It will consist of a large amount of pictures presented as gallery that the user can view in order to be provided with a visual representation of the features of the resort. This page will contain images (.jpeg, .gif, .png), HTML form, CSS for styling, hyperlinks to other pages, Html table, javascript.
