Schedule Page Details

The schedule page will allow guests of the website to schedule a stay at the resort. At this point in the process, the guest has already selected a room and is ready to select when and for how long they will stay at the resort. The schedule page allows guests to carry out this portion of the process. This page will contain images (.jpeg, .gif, .png), radio buttons and checkboxes, HTML form, CSS for styling, hyperlinks to other pages, Html table, javascript. This page will not feature video, flash or shockwave, java applet, hyperlink to an image, text formatting and text breaks or html frames.
