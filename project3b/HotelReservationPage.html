Hotel Reservations Page Details

The hotel reservations page will allow guests of the website to search the rooms offered at the resort and select the room of their choice. There will be a multitude of rooms to select from and these rooms will vary in price and features such as the number of beds, kitchens, ocean views, giving users plenty of flexibility in regards to options to pick from. Information on the specific details of each and every room will also be offered on this page. This page will contain images (.jpeg, .gif, .png), radio buttons and checkboxes, HTML form, CSS for styling, hyperlinks to other pages, javascript, hyperlinks through images. This page will not feature video, html table, flash or shockwave, java applet, text formatting, text formatting and text breaks, or html frames.
